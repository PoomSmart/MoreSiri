MoreSiri
=============

More languages support as of iOS 8.3 beta 2+ for earlier OS Siri.

These languages were added:

From iOS 8.3
===========
* en-SG = English for Singapore
* pt-BR = Portuguese for Brazil
* da-DK = Danish
* nl-NL = Dutch
* en-NZ = English for New Zealand
* en-IN = English for India
* ru-RU = Russian
* sv-SE = Swedish
* th-TH = Thai
* tr-TR = Turkish
